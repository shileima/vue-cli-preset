### 项目目录

```javascript
├── public             // 模板html文件
├── vue.config.js      // vue-cli3构建配置
├── package.json       // 项目配置文件
├── src                // 生产目录
│   ├── assets         // 静态资源(图片、css)
│   ├── components     // 全局通用组件
│   ├── views          // 各页面入口，路由级别组件
│   ├── router         // 路由相关配置目录
│   ├── service        // 后端接口API
│   ├── store          // vuex相关文件
│   └── App.vue        // 统一入口vue文件
│   └── main.js        // Webpack 预编译入口
```
